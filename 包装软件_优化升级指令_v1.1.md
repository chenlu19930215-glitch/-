## 模块一：方案去重合并（Bug修复，最高优先级）

### 问题描述

当前方案汇总表存在重复方案：方案1-4是同一组合的不同"旋转姿态"，方案5-8同理。  
根本原因：装箱算法对纸盒6种放置姿态全部输出，导致纸箱外尺寸相同但L/W互换的方案被视为不同方案。

### 修复逻辑

在 `src/engine/optimizer.ts`（或 boxing.ts 的方案输出处）添加去重逻辑：

```typescript
// 去重规则：将纸箱外尺寸的三个维度排序后作为唯一key
// 例如：382×499×142 和 499×382×142 → 排序后都是 142×382×499 → 同一方案
function deduplicateSolutions(solutions: BoxingSolution[]): BoxingSolution[] {
  const seen = new Map<string, BoxingSolution>();
  
  for (const sol of solutions) {
    // 将外尺寸三个维度排序，生成唯一标识key
    const dims = [sol.outerDims.L, sol.outerDims.W, sol.outerDims.H].sort((a, b) => a - b);
    // key还需包含每箱数量，避免同尺寸不同装法被合并
    const key = `${dims[0]}_${dims[1]}_${dims[2]}_${sol.totalCount}`;
    
    if (!seen.has(key)) {
      seen.set(key, sol);
    } else {
      // 已存在相同方案时，保留综合评分更高的
      const existing = seen.get(key)!;
      if (sol.score > existing.score) {
        seen.set(key, sol);
      }
    }
  }
  
  return Array.from(seen.values()).sort((a, b) => b.score - a.score);
}
```

**调用位置**：在 `optimizer.ts` 返回最终结果前调用此函数，确保传给UI的方案列表已去重。

**验证**：使用图片中的参数（120×053×120mm纸盒，1200×1000×150mm托盘），方案汇总应只显示5个不重复方案，而非10个。

## 模块二：3D可视化升级为 React Three Fiber（核心功能升级）

### 当前问题

Canvas 2D等轴测渲染存在以下缺陷：
- 视角固定，无法旋转/缩放
- 托盘视图被裁切（当前图片右侧托盘视图只能看到顶部角落）
- 纸箱视图缺少纸盒内部分层颜色区分
- 无交互，用户无法验证排列是否正确

### 技术方案

用 **React Three Fiber + @react-three/drei** 替换现有Canvas 2D渲染器。

#### 依赖安装

```bash
pnpm add @react-three/fiber @react-three/drei three
pnpm add -D @types/three
```

#### 纸箱内部视图组件

新建 `src/components/visualization/CartonView3D.tsx`：

```tsx
import { Canvas } from '@react-three/fiber'
import { OrbitControls, Box, Text } from '@react-three/drei'
import { useMemo } from 'react'
import type { BoxingSolution } from '../../engine/types'

interface CartonView3DProps {
  solution: BoxingSolution
  cartonDims: { L: number; W: number; H: number }
}

// 颜色方案：参考原始设计图
const CARTON_COLORS = ['#E8567A', '#4A9FD4', '#6BBF59', '#F5A623']
const BOX_WALL_COLOR = '#C8B48A'   // 瓦楞纸色
const BOX_WALL_OPACITY = 0.15       // 半透明，能看到内部

export function CartonView3D({ solution, cartonDims }: CartonView3DProps) {
  const { countL, countW, countH, innerDims } = solution
  
  // 计算每个纸盒的3D位置
  const cartons = useMemo(() => {
    const items = []
    const cl = cartonDims.L / 1000  // 转换为Three.js单位（米）
    const cw = cartonDims.W / 1000
    const ch = cartonDims.H / 1000
    
    for (let l = 0; l < countL; l++) {
      for (let w = 0; w < countW; w++) {
        for (let h = 0; h < countH; h++) {
          items.push({
            position: [
              (l + 0.5) * cl - (countL * cl) / 2,
              (h + 0.5) * ch - (countH * ch) / 2,
              (w + 0.5) * cw - (countW * cw) / 2,
            ] as [number, number, number],
            color: CARTON_COLORS[h % CARTON_COLORS.length],
            size: [cl * 0.97, ch * 0.97, cw * 0.97] as [number, number, number],
          })
        }
      }
    }
    return items
  }, [solution, cartonDims])
  
  // 纸箱外壁尺寸
  const boxW = innerDims.L / 1000
  const boxH = innerDims.H / 1000
  const boxD = innerDims.W / 1000

  return (
    <div style={{ width: '100%', height: '360px', background: '#F8F9FA', borderRadius: 8 }}>
      <Canvas camera={{ position: [0.8, 0.6, 0.8], fov: 45 }}>
        <ambientLight intensity={0.6} />
        <directionalLight position={[3, 5, 3]} intensity={0.8} />
        <directionalLight position={[-2, 3, -2]} intensity={0.3} />
        
        {/* 纸盒 */}
        {cartons.map((c, i) => (
          <mesh key={i} position={c.position}>
            <boxGeometry args={c.size} />
            <meshLambertMaterial color={c.color} />
          </mesh>
        ))}
        
        {/* 纸箱外壁（半透明） */}
        <mesh position={[0, 0, 0]}>
          <boxGeometry args={[boxW, boxH, boxD]} />
          <meshLambertMaterial 
            color={BOX_WALL_COLOR} 
            transparent 
            opacity={BOX_WALL_OPACITY}
            side={2}  // DoubleSide
          />
        </mesh>
        
        {/* 轴标签 */}
        <Text position={[boxW / 2 + 0.05, 0, 0]} fontSize={0.04} color="#333">L</Text>
        <Text position={[0, boxH / 2 + 0.05, 0]} fontSize={0.04} color="#333">H</Text>
        <Text position={[0, 0, boxD / 2 + 0.05]} fontSize={0.04} color="#333">W</Text>
        
        <OrbitControls enablePan={true} enableZoom={true} enableRotate={true} />
        <gridHelper args={[2, 20, '#DDD', '#DDD']} position={[0, -boxH/2, 0]} />
      </Canvas>
      <div style={{ textAlign: 'center', fontSize: 12, color: '#666', padding: '4px 0' }}>
        拖拽旋转 · 滚轮缩放 · 右键平移
      </div>
    </div>
  )
}
```

#### 托盘码垛视图组件

新建 `src/components/visualization/PalletView3D.tsx`：

```tsx
import { Canvas } from '@react-three/fiber'
import { OrbitControls } from '@react-three/drei'
import { useMemo } from 'react'
import type { PalletSolution } from '../../engine/types'

const LAYER_COLORS = ['#6B8C3A', '#3D4A5C']  // 奇偶层交替色
const PALLET_COLOR = '#D4B854'                 // 托盘黄棕色
const PALLET_LEG_COLOR = '#B8963E'

interface PalletView3DProps {
  solution: PalletSolution
  palletDims: { L: number; W: number; H: number }
}

export function PalletView3D({ solution, palletDims }: PalletView3DProps) {
  const { layerCountL, layerCountW, layers, boxingSolution } = solution
  const boxOD = boxingSolution.outerDims
  
  // 单位统一转换为Three.js（/1000转米，再*3缩放便于显示）
  const scale = 3 / Math.max(palletDims.L, palletDims.W) * 1000
  
  const pl = palletDims.L / 1000 * scale
  const pw = palletDims.W / 1000 * scale
  const ph = palletDims.H / 1000 * scale
  const bl = boxOD.L / 1000 * scale
  const bw = boxOD.W / 1000 * scale
  const bh = boxOD.H / 1000 * scale
  
  const boxes = useMemo(() => {
    const items = []
    for (let layer = 0; layer < layers; layer++) {
      for (let lIdx = 0; lIdx < layerCountL; lIdx++) {
        for (let wIdx = 0; wIdx < layerCountW; wIdx++) {
          items.push({
            position: [
              (lIdx + 0.5) * bl - (layerCountL * bl) / 2,
              ph + (layer + 0.5) * bh,
              (wIdx + 0.5) * bw - (layerCountW * bw) / 2,
            ] as [number, number, number],
            color: LAYER_COLORS[layer % 2],
          })
        }
      }
    }
    return items
  }, [solution, palletDims, scale])

  return (
    <div style={{ width: '100%', height: '400px', background: '#F0F2F5', borderRadius: 8 }}>
      <Canvas camera={{ position: [2.5, 2, 2.5], fov: 45 }}>
        <ambientLight intensity={0.5} />
        <directionalLight position={[4, 6, 4]} intensity={0.9} castShadow />
        <directionalLight position={[-3, 4, -3]} intensity={0.25} />
        
        {/* 托盘板面 */}
        <mesh position={[0, ph / 2, 0]}>
          <boxGeometry args={[pl, ph, pw]} />
          <meshLambertMaterial color={PALLET_COLOR} />
        </mesh>
        
        {/* 托盘腿（3×3排列） */}
        {[-1, 0, 1].map(xi => [-1, 0, 1].map(zi => (
          <mesh key={`${xi}_${zi}`} position={[xi * pl / 3, ph * 0.3, zi * pw / 3]}>
            <boxGeometry args={[pl / 5, ph * 0.6, pw / 5]} />
            <meshLambertMaterial color={PALLET_LEG_COLOR} />
          </mesh>
        )))}
        
        {/* 纸箱 */}
        {boxes.map((b, i) => (
          <mesh key={i} position={b.position}>
            <boxGeometry args={[bl * 0.98, bh * 0.98, bw * 0.98]} />
            <meshLambertMaterial color={b.color} />
          </mesh>
        ))}
        
        <OrbitControls enablePan={true} enableZoom={true} enableRotate={true} />
        <gridHelper args={[4, 40, '#CCC', '#CCC']} position={[0, 0, 0]} />
      </Canvas>
      <div style={{ textAlign: 'center', fontSize: 12, color: '#666', padding: '4px 0' }}>
        拖拽旋转 · 滚轮缩放 · 右键平移
      </div>
    </div>
  )
}
```

### 6.2 Nginx 配置文件

在项目根目录创建 `nginx.conf`（供服务器配置参考）：

```nginx
server {
    listen 80;
    listen 443 ssl;
    server_name your-domain.com;  # 替换为你的域名或服务器IP

    # SSL证书（阿里云免费证书路径，按实际修改）
    # ssl_certificate /etc/nginx/ssl/cert.pem;
    # ssl_certificate_key /etc/nginx/ssl/key.pem;

    index index.html;

    # 启用gzip压缩（Three.js chunk较大，压缩效果明显）
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml application/xml image/svg+xml;
    gzip_min_length 1024;
    gzip_comp_level 6;

    # 静态资源长期缓存（带hash文件名）
    location ~* \.(js|css|png|jpg|woff2|woff)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # SPA路由支持（React Router）
    location / {
        try_files $uri $uri/ /index.html;
        add_header Cache-Control "no-cache";
    }

    # 安全headers
    add_header X-Frame-Options SAMEORIGIN;
    add_header X-Content-Type-Options nosniff;
    add_header X-XSS-Protection "1; mode=block";
}
```

### 6.3 部署脚本

```bash

echo "📦 上传到服务器..."

  -e "ssh -i $SSH_KEY" \
  ./dist/ \
  $SERVER_USER@$SERVER_IP:$DEPLOY_PATH/dist/

echo "🔄 重载Nginx..."
ssh -i $SSH_KEY $SERVER_USER@$SERVER_IP "nginx -s reload"

echo "✅ 部署完成！"
echo "访问地址：http://$SERVER_IP"
```

### 6.4 Docker 部署方案（推荐，更稳定）

```dockerfile

RUN pnpm build

# 运行阶段（仅Nginx + 静态文件）

COPY --from=builder /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf

CMD ["nginx", "-g", "daemon off;"]
```

```yaml
version: '3.8'
services:

    build: .
    ports:

      - "443:443"
    restart: unless-stopped
    volumes:

      - ./ssl:/etc/nginx/ssl:ro
```

## 注意事项

1. **Three.js 包体积**：三个包合计约 600KB（gzip后约 180KB），需确保 vite.config.ts 的 manualChunks 配置正确，避免主包过大导致首屏慢
2. **OrbitControls 阻止页面滚动**：3D视图容器设置 `touch-action: none` 避免移动端滚动冲突
3. **去重逻辑位置**：必须在 optimizer 层做，而不是在 UI 层过滤，保证数据一致性
4. **部署时 base path**：如果服务器部署到根路径（如 `http://ip/`），vite.config.ts 中 base 保持默认；如果部署到子路径（如 `http://ip/tools/`），需设置 `base: '/tools/'`
